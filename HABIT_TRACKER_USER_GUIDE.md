# 🎯 Habit Tracker — Guide d'utilisation

## Vue d'ensemble

Le suivi d'habitudes vous permet de construire de nouveaux comportements (habitudes à acquérir) ou d'éliminer des comportements indésirables (habitudes à éliminer). L'objectif par défaut est de **60 jours consécutifs** — un délai scientifiquement prouvé pour ancrer une habitude.

## 🚀 Démarrage rapide

### 1. Créer une habitude

1. Cliquez sur l'onglet **🎯 Habitudes**
2. Cliquez **+ Nouvelle habitude**
3. Remplissez le formulaire:
   - **Nom**: Ex: "Méditer 10 minutes", "Lire 30 pages", "Éviter les réseaux sociaux le matin"
   - **Type**: 
     - 🟢 **Acquérir** — pour construire une nouvelle habitude
     - 🔴 **Éliminer** — pour arrêter un comportement
   - **Département**: Liez à un département (Résilience, Opérations, etc.) ou laissez vide
   - **Objectif**: Nombre de jours (défaut: 60)
   - **Manqués autorisés**: Marge d'erreur (ex: 3 = vous pouvez manquer 3 jours sans briser votre série)
   - **Icône**: Emoji de votre choix (ex: 🧘, 📖, 🚫)
4. Cliquez **Enregistrer**

### 2. Logger rapidement (friction minimale)

**Méthode 1: Dashboard du jour (recommandée)**
- Dans la section **📅 Aujourd'hui**, cliquez sur le bouton de l'habitude
- ✅ Fait! Pas de formulaire, pas de friction

**Méthode 2: Bouton ✓ Log**
- Cliquez sur **✓ Log** à côté de l'habitude dans la liste
- Ouvre un formulaire pour ajouter des détails (optionnel)

### 3. Ajouter des détails (optionnel)

Si vous voulez capturer plus de contexte:
- **Note**: Observations, réflexions, contexte
- **Temps passé**: Durée en minutes
- **Temp**: Score d'humeur/énergie (1-10, comme le journal)
- **Date/heure**: Pour enregistrer rétroactivement (backdating)
- **Statut**: ✅ Fait ou ❌ Manqué (pour le logging négatif)

## 📊 Suivi et statistiques

### Série (Streak)
- **Nombre de jours consécutifs** où vous avez complété l'habitude
- Indicateur 🔥 quand la série est active
- Réinitialisation à 0 si vous manquez un jour (sauf si vous avez des "manqués autorisés")

### Grille 7 derniers jours
- **Cases colorées** = jour complété
- **Cases grises** = jour manqué
- Vision rapide de votre constance

### Barre de progression
- **Pourcentage** vers votre objectif (ex: 23/60 jours = 38%)
- **Couleur** du département pour identification rapide

### Completion du jour
- **% d'habitudes complétées aujourd'hui** (ex: 3/5 = 60%)
- Affiché en haut de la section Aujourd'hui

## 🎯 Stratégies recommandées

### Habitudes à acquérir (🟢)
**Exemples:**
- Méditer 10 minutes le matin
- Lire 30 pages par jour
- Faire 20 minutes d'exercice
- Écrire 500 mots
- Appeler un proche par semaine

**Conseils:**
- Commencez petit (10 min > 60 min)
- Liez à un déclencheur existant (ex: "Après le café du matin")
- Utilisez des "manqués autorisés" pour les habitudes non quotidiennes (ex: sport 5x/semaine = 2 manqués autorisés)

### Habitudes à éliminer (🔴)
**Exemples:**
- Ne pas consulter les réseaux sociaux avant 10h
- Éviter le sucre raffiné
- Ne pas vérifier les emails le soir
- Arrêter de fumer

**Conseils:**
- Logger la **réussite** (j'ai tenu) plutôt que l'échec
- Utiliser le logging négatif pour analyser les déclencheurs d'échec
- Remplacer par une habitude positive (ex: marcher au lieu de fumer)

## 🔥 Maintenir la série

### Option 1: Série stricte (défaut)
- **Manqués autorisés: 0**
- Un seul jour manqué = série réinitialisée
- Bon pour les habitudes critiques (médication, etc.)

### Option 2: Série flexible
- **Manqués autorisés: 2-3**
- Permet des exceptions (maladie, voyage)
- Bon pour les habitudes 5x/semaine
- Exemple: Objectif 60 jours, 3 manqués = vous pouvez manquer 3 jours sans perdre votre série

### Logging négatif (optionnel)
- Utilisez **❌ Manqué** pour capturer les échecs
- **Avantage:** Analyse des patterns (pourquoi j'ai manqué?)
- **Inconvénient:** Peut être démotivant si trop fréquent
- **Recommandation:** Utilisez pour apprendre, pas pour punir

## 📅 Backdating (enregistrement rétroactif)

Si vous oubliez de logger:
1. Cliquez **✓ Log**
2. Modifiez la **Date & heure**
3. Enregistrez

⚠️ **Note:** Le backdating est autorisé mais découragé — l'objectif est de créer une routine de logging immédiate.

## 🗂 Organisation par département

Les habitudes peuvent être liées à vos départements existants:
- **🌱 Résilience**: Santé, sport, méditation, sommeil
- **⚙️ Opérations**: Routines administratives, finances, organisation
- **📈 Ventes & Marketing**: Réseautage, prospection, contenu
- **🏭 Production**: Création, livraison, compétences

**Avantage:** Filtrez par département pour voir les habitudes liées à un domaine spécifique.

## 🎓 Principes de formation d'habitudes

### La règle des 60 jours
- **Recherche:** Il faut en moyenne 66 jours pour ancrer une habitude (étude UCL)
- **Notre cible:** 60 jours = objectif ambitieux mais réaliste
- **Après 60 jours:** L'habitude devient automatique (moins d'effort conscient requis)

### Friction minimale
- **Principe:** Plus c'est facile, plus vous le ferez
- **Application:** Un clic pour logger = pas d'excuse
- **Résultat:** Constance > perfection

### La puissance des séries
- **Effet psychologique:** Ne pas briser la chaîne
- **Motivation:** Plus la série est longue, plus vous voulez la maintenir
- **Danger:** Ne pas devenir obsédé — la flexibilité est OK (voir "manqués autorisés")

## ❌ Archiver une habitude

Quand une habitude est acquise (ou abandonnée):
1. Cliquez sur l'habitude pour l'éditer
2. Cliquez **🗃 Archiver**
3. L'habitude disparaît de la liste active mais reste dans la base de données

**Note:** Archiver ≠ Supprimer. Les données historiques sont préservées.

## 🆘 Dépannage

**Je ne vois pas mes habitudes**
- Vérifiez le filtre de département (cliquez "Tous" pour tout voir)
- Vérifiez que l'habitude n'est pas archivée

**Ma série n'augmente pas**
- Vérifiez que vous loggez avec `completed: true` (✅ Fait)
- Vérifiez la date de logging (pas dans le futur)

**J'ai manqué un jour, ma série est cassée**
- C'est normal si "manqués autorisés" = 0
- Recommencez à 0 — c'est une opportunité d'apprendre
- Ajustez "manqués autorisés" si vous voulez plus de flexibilité
